7 Concluding Remarks

Remark 15

All results hold for subsets of an arbitrary field of characteristic \(p\) instead of \(\mathbb {Z}_p\), with the same proof.

Remark 16

Theorem 3.3 implies that if \(A\) is a subset of \(\mathbb {Z}_p\) and \(|A|\geq (p+s^{2}-1)/s\), then \(s^{\wedge }A=\mathbb {Z}_p\). This can be used to construct explicit codes for write-once memories.
